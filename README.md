# Assignment 1: Data Audit and Exploratory Data Analysis (EDA)

## Overview

This assignment introduces the fundamental process of understanding, evaluating, and exploring a dataset before applying statistical or machine learning techniques. Each student will be assigned a unique dataset and is expected to perform a comprehensive data audit and exploratory data analysis (EDA) using Python.

The objective is to develop the ability to identify data quality issues, summarize dataset characteristics, generate meaningful visualizations, and communicate key insights effectively.

## Learning Objectives

By completing this assignment, students will be able to:

- Assess the structure and quality of a dataset.
- Identify missing values, duplicates, inconsistencies, and outliers.
- Compute and interpret descriptive statistics.
- Create informative visualizations using Python.
- Draw meaningful conclusions from exploratory analysis.
- Communicate analytical findings in a concise professional report.

## Deliverables

Students must submit:

1. A PDF report (2–3 pages).
2. A Jupyter Notebook (`.ipynb`) containing all analysis code.
3. The dataset used for the assignment.

## Suggested Analysis Tasks

The analysis should include, where applicable:

- Dataset overview and variable descriptions.
- Data type verification.
- Missing value assessment.
- Duplicate record identification.
- Summary statistics.
- Distribution analysis.
- Outlier detection.
- Relationship exploration between variables.
- Appropriate visualizations.
- Key findings and recommendations.

## Tools

Students are expected to use Python and may utilize libraries such as:

- Pandas
- NumPy
- Matplotlib
- Seaborn

Additional libraries may be used if appropriately justified.

## Academic Integrity

Each student receives a unique dataset. All submitted work must represent the student's own analysis, code, visualizations, and written interpretation. Students should be prepared to explain their work during follow-up discussions or presentations.
